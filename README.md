# README

## General info
This project is Sabelle Beaute - An ecommerce beauty cosmetic site with all natural and homemade products.
	
## Technologies
Project is created with:
* Rails version: 2.74
* React version: 2.33
* postgreSQL database
	
## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
